# Olist Marketplace Logistics Analysis

## Project Overview

This project analyzes logistics and delivery performance in the Brazilian e-commerce marketplace Olist.

The goal is to identify operational inefficiencies and understand how delivery performance affects the overall customer experience.

## Dataset

Dataset: Olist Brazilian E-commerce Public Dataset

Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset contains information about:

- Orders
- Sellers
- Customers
- Products
- Payments
- Delivery performance

Total records: ~100k orders.

## Tools Used

SQL (BigQuery) – data extraction and analysis

Tableau – data visualization and dashboard creation

GitHub – project documentation

## Key Metrics

- Total Orders
- Total Revenue
- Average Shipping Time
- Late Delivery Rate
- Freight Cost by State

## Key Insights

1. Late delivery rate fluctuated significantly across months, reaching over 20% in early 2018.

2. Furniture-related categories experienced the longest delivery times due to logistics complexity.

3. Freight costs vary significantly between Brazilian states, indicating regional logistics challenges.

4. A small number of sellers generated a large proportion of total revenue.
